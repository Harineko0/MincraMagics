# MincraMagics

Minecraftに魔法を追加するSpigotプラグインです。  
[MincraMagicRod](https://github.com/celtas/MincraMagicRod)の後継になります。


## 前提プラグイン

NBT-API (https://github.com/tr7zw/Item-NBT-API)
